# OpenHouse AI - Marketing Site

Marketing website for the OpenHouse AI Platform.

## Overview

The Marketing Site will serve as the public face of OpenHouse AI, featuring:
- Product overview and features
- Pricing information
- Customer testimonials
- Contact and demo request forms
- Blog and resources

## Development

```bash
# From the marketing directory
npm run dev

# From the root directory
npm run dev:marketing
```

The app runs on http://localhost:3003 by default.

## Status

This is currently a placeholder marketing shell created as part of the monorepo restructuring.
The current implementation includes a basic landing page showcasing the platform's core features.
