Run the SQL migration first - creates these tables:

custom_qa - Development-specific Q&A overrides
knowledge_base - Additional knowledge per development
question_analytics - Track all questions for insights
platform_insights - AI-generated insights storage

Feature: Assistant Training Portal
4 Tabs:

Test Assistant - Live chat to test responses per development
Custom Q&A - Add question/answer pairs that override AI
System Instructions - Custom prompts per development
Knowledge Base - Add facts/info the AI can reference

How it works:

Select development from dropdown
Custom Q&As take priority over AI responses
System instructions prepended to all prompts
Test changes immediately in Test tab

Feature: R&D Analytics Dashboard
5 Sections:

Overview - Topic breakdown with trends
Trending Questions - Most asked nationwide
Regional Breakdown - Activity by county
Knowledge Gaps - Low-confidence topics
All Insights - AI-generated recommendations

Key Metrics:

Total questions nationwide
Avg questions per unit
Satisfaction rate
Topic breakdown with trends
Regional comparison

Environment Variables
Ensure ANTHROPIC_API_KEY is set for the test assistant feature.
