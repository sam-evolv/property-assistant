Flow

From Submissions page → Click "Create Development" → Pre-fills form with submission data
Step 1: Details → Name, address, tenant, units estimate
Step 2: Branding → Drag-drop 3 logos (sidebar, assistant, toolbar)
Step 3: AI Config → Custom system instructions
Step 4: Import Units → Upload spreadsheet, map columns, auto-generate access codes (format: LVP-001-X7K2)
Step 5: Complete → Success + links to manage

Access Code Format
{DEV_PREFIX}-{UNIT_NUM}-{RANDOM4}

DEV_PREFIX: First letters of development name (e.g., "Longview Park" → "LVP")
UNIT_NUM: 3-digit padded number (001, 002, etc.)
RANDOM4: 4 random alphanumeric chars (no 0/O/1/I to avoid confusion)

Example: LVP-001-A2X9, LVP-002-K7M3
